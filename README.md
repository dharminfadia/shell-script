# shell-read-file
A shell script to read file line by line


Note: In the input file you need to have a new line ends with. See `input.txt`.

# Run script
```
cd path_to_shell-read-file/
chmod u+x read.sh
```

# With no input
```
./read.sh 
```
The above will throw error

# With invalid input
```
./read.sh dummy
```
The above will throw error

# With valid input
```
./read.sh ./input.txt
```

The above will display the data
